How invite links work

Share the invite link with the person you want to add. When they visit the link and log in via Authentik, their role is automatically set.

Single use — link works once then burns. Best for individual invites.

Multi-use — link works for multiple people. Set a max count or leave unlimited. Add an expiry date for time-limited access (e.g. a class registration period).

Revoking a link prevents future claims but does not change roles already granted.
