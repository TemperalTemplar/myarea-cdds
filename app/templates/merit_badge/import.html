Import Merit Badge Worksheet

Upload a BSA merit badge workbook (PDF or DOCX). The system will extract all numbered requirements and create a course draft ready to edit and publish.

How it works:
Upload the official BSA workbook PDF or DOCX
System extracts all requirements automatically
Each requirement becomes a lesson module
Review, edit, and publish to your CDDS catalog
LMS nodes can ingest and assign to Scouts
Accepted: PDF, DOCX · Official BSA workbooks from usscouts.org or meritbadge.org
Parsing runs on your local GPU via Ollama — no external API calls, fully sovereign.
